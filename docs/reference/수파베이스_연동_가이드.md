## 3. MCP (AI 연동) 설정 방법
재부팅 후 Trae IDE를 켜면 MCP 설정 창에서 다음 과정을 진행하세요.

### **[수동 추가(Add Manually) 시 사용할 JSON 코드]**
설정 창의 중괄호(`{}`) 안에 아래 내용을 복사해서 넣으시면 됩니다.

{
  "mcpServers": {
    "supabase": {
      "command": "npx",
      "args": ["-y", "@supabase/mcp-server-supabase"],
      "env": {
        "SUPABASE_URL": "https://<your-project>.supabase.co",
        "SUPABASE_SERVICE_ROLE_KEY": "<your-service-role-key>",
        "SUPABASE_ACCESS_TOKEN": "<your-access-token>"
      }
    }
  }
}



## 4. 재부팅 후 해결 순서 (Troubleshooting)

1. **기존 서버 삭제**: 목록에 `supabase`나 `supabase-mcp`가 빨간색으로 떠 있다면, 휴지통 아이콘을 눌러 **모두 삭제**하세요.
2. **새로 추가**: `+ Add` > `Add Manually`를 선택하고 위 3번의 JSON 코드를 붙여넣으세요.
3. **브라우저 승인 (가장 중요!)**: 
   - `Retry` 버튼을 누른 후, 작업 표시줄의 **브라우저(크롬, 엣지 등)**를 확인하세요.
   - `Authorize LinkDrop` 또는 `Sign in to Supabase` 창이 떠 있을 것입니다.
   - 반드시 **[Authorize]** 버튼을 눌러 승인을 완료해야 합니다.
4. **연결 확인**: Trae IDE에서 초록색 불이 들어오는지 확인합니다.
