## 2. 데이터-노드 매핑 규칙 (Data-to-Node Mapping)
`VideoScene` 객체의 각 필드는 Opal 엔진의 다음 노드들과 1:1로 매핑됩니다.

| VideoScene 필드 | Opal 대응 노드 | 처리 로직 (Logic) |
| :--- | :--- | :--- |
| **`text`** | `TextNode` | 시니어 표준 폰트(Gmarket Sans) 및 크기(72pt+) 적용. |
| **`keywords`** | `AssetNode` | 키워드 기반의 배경 영상/이미지 자동 검색 및 매칭. |
| **`mood`** | `GradeNode` | 무드에 따른 LUT(색감) 및 **효과음(SE) 테마** 결정. |
| **`duration`** | `TimelineNode` | 장면의 시작/종료 시간 및 전환 효과(Transition) 길이 제어. |

---

## 3. Opal 브릿지 아키텍처 (Opal Bridge)
백엔드(`apps/api`)는 다음 과정을 통해 Opal과 통신합니다.

1.  **Request**: `data_transformer`가 생성한 JSON을 Opal API 규격으로 변환.
2.  **Auth**: 전용 API Key 또는 세션 토큰을 통해 보안 연결 확보.
3.  **Trigger**: Opal 엔진에 '렌더링 시작' 신호 송신.
4.  **Callback**: 영상 조각 생성 완료 시 결과물 경로(URL/Path) 수신.

## 4. 시니어 특화 노드 설정 (Senior-Specific Nodes)
시니어 사용자의 편안한 시청을 위해 Opal 노드에 다음 설정을 강제합니다. (Python v1.2.0 엔진 반영 완료)

### ① AudioNode: 청각 마스터링 규격
- **BGM(배경음악)**: **`bgm_volume: 0.0`** (나레이션 집중도를 위해 완전 배제)
- **SE(효과음) 트리거**: 문장 성격에 따라 3가지 테마 자동 매칭
    - `scene_open_whoosh`: 장면 전환 시 부드러운 주의 환기
    - `question_ding`: 질문형 문장에서 집중도 유도
    - `highlight_pop`: 강조 문구 등장 시 신뢰감 부여
- **TTS**: `ko-KR-Wavenet-D` (차분하고 명확한 톤 고정)

### ② MotionNode: 안정적 시각 효과
- **Effect**: **`Ken-Burns-Zoom`** (아주 천천히 안으로 빨려 들어가는 줌 효과)
- **Shake**: **`intensity: 0.0`** (카메라 흔들림 절대 금지 - 멀미 방지)

### ③ TextNode: 가독성 극대화
- **Font**: `Gmarket Sans` (굵고 명확한 고딕체)
- **Safe Area**: 하단 15% 지점 고정
- **Background Bar**: **`opacity: 0.4`** 블랙 반투명 바 필수 생성 (텍스트 뒤)
