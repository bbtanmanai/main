# 115 SVG 비디오용 캐릭터 표준화 가이드

> 작성일: 2026-03-18
> 버전: v3.0
> 적용 범위: Remotion 영상 제작용 캐릭터 전체

---

## 1. 개요

Remotion 영상에 사용되는 캐릭터의 파츠 분리 규격, 그리드 생성, 좌표 캘리브레이션, SVG 애니메이션 표준을 정의한다.

AI로 생성한 대표이미지 1장 + 그리드 이미지 4장으로 파츠 PNG를 확보하고,
캐릭터별 좌표 JSON으로 조립하여 SVG 애니메이션을 구현한다.

---

## 2. 핵심 원칙

- **소스 5장**: 대표이미지 + mouth_Grid + eyes_Grid + head_Grid + arm_Grid
- **파츠 6종**: body(대표이미지 크롭 1장, CSS rotate로 기울기) + head(3) + mouth(3) + eyes(3) + arm(4)
- **흰색 아웃라인 표준**: 모든 캐릭터/파츠에 흰색 스티커 컷아웃 아웃라인 적용 (배경 독립성 + 브랜드 통일)
- **캐릭터별 좌표 JSON**: crop, anchor, pivot, on_head 좌표로 조립
- **SVG 애니메이션**: CSS transform 기반, Remotion 프레임 제어
- **립싱크**: TTS SRT 타이밍 연동
- **파일명 접두사**: `c{N}_mouth_Grid.png` 형식

---

## 3. 전체 흐름

```
[대표이미지] c{N}.png (9:16)
      ↓ 레퍼런스 첨부하여 AI 생성
c{N}_mouth_Grid.png   (16:9, 3뷰)  — 입 모양 3종
c{N}_eyes_Grid.png    (16:9, 3뷰)  — 눈 상태 3종
c{N}_head_Grid.png    (16:9, 3뷰)  — 머리 방향 3종
c{N}_arm_Grid.png     (16:9, 4뷰)  — 팔 동작 4종
※ body는 대표이미지에서 크롭 (별도 Grid 불필요, CSS rotate로 기울기 표현)
      ↓
[캘리브레이션] 프로그램 GUI로 파츠 좌표 지정 → JSON 저장
      ↓
[슬라이싱] JSON 좌표 기반 → 파츠 PNG 출력
      ↓
[Remotion 렌더링]
  body(대표이미지 크롭) + head + mouth + eyes + arms
  → CSS transform 애니메이션 + TTS 립싱크
```

### 립싱크 (TTS 연동)

```
SRT 타이밍:
  00:00:00,100 → 00:00:04,320  "칠순 잔치 날, 아버지께서..."

음성 구간:
  mouth_closed (기본)
  → 텍스트 시작 → mouth_half/mouth_open 교차 (0.15초 간격)
  → 텍스트 종료 → mouth_closed

무음 구간:
  mouth_closed (고정)
```

## 7. AI 생성 프롬프트 (최종 v3.0)

> 생성 순서: ① 대표이미지 → ② mouth_Grid → ③ eyes_Grid → ④ head_Grid → ⑤ arm_Grid
> 각 그리드 생성 시 **대표이미지를 반드시 레퍼런스로 첨부**.
> 프롬프트는 **범용** — 캐릭터별 특징은 레퍼런스 이미지가 결정.

### 7-1. mouth_Grid 프롬프트

```
Create a single 16:9 reference sheet image containing exactly 3 panels of the uploaded character's mouth area, arranged in a horizontal row.

=== CANVAS ===
- Final image ratio: 16:9 (e.g., 1920×1080 pixels)
- Solid uniform flat background: #808080 (medium gray)
- NO gradients, NO textures, NO patterns in the background
- NO borders, NO dividing lines, NO gaps, NO frames between panels
- The 3 panels should seamlessly sit side by side with only the background separating them

=== PANEL LAYOUT ===
- Exactly 3 panels. NOT 2, NOT 4. Exactly THREE.
- Each panel occupies exactly 1/3 of the total canvas width
- All 3 panels share the EXACT same:
  • Zoom level (identical magnification)
  • Vertical crop position (identical Y-axis framing)
  • Horizontal centering (face centered in each panel)
  • Lighting and color grading

=== CROP / FRAMING ===
- Camera frames from JUST BELOW THE EYES down to the CHIN
- Visible areas: lower cheeks, nose tip, nasolabial folds, mouth, chin, jawline
- The very top of the frame may show the bottom edge of the eyes (this is acceptable)
- DO NOT show forehead, hair, or ears
- DO NOT crop too tight (chin must be fully visible with some margin below)

=== THE 3 MOUTH STATES ===
Panel 1 (LEFT):   CLOSED — Mouth gently closed, lips together, natural resting expression.
                   No tension in the jaw. Relaxed, neutral.
Panel 2 (CENTER): HALF OPEN — Mouth slightly parted, as if mid-word during speech.
                   Upper and lower teeth may be barely visible. Lips relaxed but separated.
Panel 3 (RIGHT):  FULLY OPEN — Mouth clearly open wide, as if pronouncing "아" or "오".
                   Upper teeth, lower teeth, and tongue clearly visible. Active speaking pose.

=== STYLE PRESERVATION (LinkDrop Standard) ===
- MANDATORY: Apply a clean WHITE OUTLINE (sticker cut-out border) around the character/part
  • Outline width: approximately 3~5px at 1920×1080 scale
  • Outline color: pure white (#FFFFFF)
  • Outline must be smooth, consistent, and follow the character silhouette precisely
  • This is a LinkDrop brand standard applied to ALL characters regardless of art style
- You MUST exactly replicate the art style, line quality, color palette, and rendering technique of the uploaded reference image
- If the reference is a cutout illustration → output must be cutout illustration
- If the reference is 3D render → output must be 3D render
- If the reference is anime → output must be anime
- DO NOT switch styles (e.g., do not make a cartoon reference look realistic)
- Skin tone, shading style, and line weight must be identical to the reference

=== ABSOLUTE PROHIBITIONS ===
- NO text, labels, numbers, or watermarks anywhere
- NO panel borders, frames, or dividing lines
- NO extra panels (must be exactly 3)
- NO full face (only lower face area)
- NO style change from the reference image
```

### 7-2. eyes_Grid 프롬프트

```
Create a single 16:9 reference sheet image containing exactly 3 panels of the uploaded character's eye area, arranged in a horizontal row.

=== CANVAS ===
- Final image ratio: 16:9 (e.g., 1920×1080 pixels)
- Solid uniform flat background: #808080 (medium gray)
- NO gradients, NO textures, NO patterns in the background
- NO borders, NO dividing lines, NO gaps, NO frames between panels

=== PANEL LAYOUT ===
- Exactly 3 panels. NOT 2, NOT 4. Exactly THREE.
- Each panel occupies exactly 1/3 of the total canvas width
- All 3 panels share the EXACT same zoom level, crop position, and centering

=== CROP / FRAMING ===
- Camera frames from the FOREHEAD (above eyebrows) down to the NOSE BRIDGE
- Visible areas: forehead (partial), eyebrows, eyes, upper nose bridge
- If character wears glasses, hat, or headband → keep them visible and consistent across all panels
- DO NOT show mouth, chin, or full hair
- DO NOT crop so tight that eyebrows are cut off

=== THE 3 EYE STATES ===
Panel 1 (LEFT):   FULLY OPEN — Eyes wide open, natural alert expression.
                   Iris and pupil clearly visible. Eyebrows in natural relaxed position.
Panel 2 (CENTER): HALF CLOSED — Eyes at 50% closed, mid-blink position.
                   Upper eyelids cover the top half of the iris. Relaxed, sleepy look.
Panel 3 (RIGHT):  FULLY CLOSED — Eyes completely shut, natural gentle blink.
                   Eyelids closed softly, no squinting or tension. Peaceful expression.

=== STYLE PRESERVATION (LinkDrop Standard) ===
- MANDATORY: Apply a clean WHITE OUTLINE (sticker cut-out border) around the character/part
  • Outline width: approximately 3~5px at 1920×1080 scale
  • Outline color: pure white (#FFFFFF)
  • Outline must be smooth, consistent, and follow the character silhouette precisely
  • This is a LinkDrop brand standard applied to ALL characters regardless of art style
- Exactly replicate the art style, line quality, color palette of the uploaded reference
- Glasses, accessories, facial features must remain identical across all 3 panels
- DO NOT switch rendering styles

=== ABSOLUTE PROHIBITIONS ===
- NO text, labels, numbers, or watermarks
- NO panel borders or dividing lines
- NO extra panels (exactly 3)
- NO mouth or chin visible
- NO style change from reference
```
