WORKFLOW

AI 연구 파트너에서 AI 아트 디렉터로

노트북LM은 단순히 문서를 요약하는 도구가 아닙니다.
'슬라이드 덱' 기능을 활용하면 당신의 이야기를 시각화하는 자동화된 스토리보드 작가가 됩니다.

STEP 01

프리 프로덕션: 제미나이로 시나리오 집필하기

  1. 구글 제미나이(Google Gemini) 접속
  2. 아래 프롬프트를 입력하여 원천 소스 생성
  3. 생성된 텍스트를 복사하여 메모장에 저장
• 프롬프트 예시

그리스 신화 판도라의 상자를 3,000자 내외의 SF 소설 형식으로 각색해줘. 배경은 실리콘 밸리로 설정해주고, 독자는 20대 이상의 성인으로 설정해줘.
STEP 02

소스 입력: 노트북LM 세팅하기

  1. 노트북LM 접속 후 'NotebookLM 사용해보기' 클릭
  2. 'Create new notebook' 버튼 클릭
  3. 소스 업로드 팝업에서 'Copied Text' 선택
  4. 저장해 둔 이야기를 붙여넣고 'Insert' 클릭
STEP 03

시각화: 슬라이드 덱 기능의 재발견

우측 'Studio' 탭의 슬라이드 덱 기능이 핵심입니다.
우리는 발표 자료를 만드는 것이 아니라, 영화의 장면(Scene)을 만듭니다.

💡 이 작은 버튼 하나가 영화 제작의 시작입니다. 아래 스타일 프롬프트를 넣어 원하는 분위기를 연출해 보세요.
STYLE PROMPTS

4가지 스타일 프롬프트 예시

🎬 Style A: 헐리우드 블록버스터 SF 단편 영화로 만들 수 있도록 각각의 슬라이드가 키프레임이 되도록 해주고, 슬라이드에는 텍스트를 넣지 말아줘. 슬라이드 스타일은 헐리우드의 블록버스터 SF 영화 스타일로 해줘.
🌸 Style B: 일본 SF 애니메이션 애니메이션으로 만들 수 있도록 각각의 슬라이드가 키프레임이 되도록 해주고, 슬라이드에는 텍스트를 넣지 말아줘. 슬라이드 그림 스타일은 일본 SF 애니메이션으로 해줘.
🖼️ Style C: 타무라 시게루 일러스트 슬라이드 그림 스타일은 타무라 시게루의 일러스트레이션으로 해줘. 컬러를 몇 가지로만 제한해서 타무라 시게루 특유의 인상적인 분위기를 연출해줘.
🖌️ Style D: 역동적인 수묵화 슬라이드 그림 스타일은 수묵화로 해주고, 붓의 힘과 속도감이 느껴지는 역동적인 필치와 농담(濃淡)의 변화를 풍부하게 표현해줘.
PRO TIPS

작업 효율을 높이는 프로 팁

Tip 01: 병렬 처리 (Parallel Processing)
하나의 슬라이드 덱 생성을 기다리지 마세요.
프롬프트를 입력해두면 백그라운드에서 동시에 여러 작업이 진행됩니다.
Tip 02: 믹스 앤 매치 (Mix & Match)
결과물이 완벽하지 않다면, 프롬프트를 약간씩 수정하여 여러 버전을 생성하세요.
여러 덱에서 가장 좋은 컷들만 골라 조합하면 완성도가 높아집니다.

• 일일 생성 용량 비교

  • Gemini Pro: 일 10~20편 (15분 대기)
  • 무료 사용자: 일 3편 (30분 대기)
STEP 04

추출: 키프레임 캡처 (Keyframe Capture)

  1. 생성된 슬라이드 우측 상단 '점 3개(...)' 클릭
  2. 'Start slideshow' 선택하여 전체 화면 모드 진입
  3. 페이지를 넘기며 결과물 확인
  4. Print Screen(스크린샷) 키로 슬라이드를 이미지 파일로 저장
💡 이렇게 저장된 이미지들은 다음 단계인 영상 생성의 '씨앗(Seed Image)'이 됩니다.
STEP 05

영상 제작: 정지된 그림에 숨결 불어넣기

캡처한 키프레임 이미지를 Image-to-Video AI 툴에 업로드하여 모션 클립을 생성합니다.

  • Veo 3 — 구글의 고품질 영상 생성 AI
  • Grok — X(트위터)의 멀티모달 AI
  • Runway — 전문가용 영상 AI
  • Luma — Dream Machine 영상 생성
STEP 06

편집: 하나의 이야기로 완성

  • 생성된 짧은 비디오 클립들을 타임라인에 배열
  • 음악, 효과음, 자막을 추가하여 최종 영상 완성
🎬 추천 편집 툴: CapCut — 무료이며 AI 자막·음악 기능 내장
