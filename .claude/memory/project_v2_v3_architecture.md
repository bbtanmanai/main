V2와 V3는 완전히 분리된 독립 프로젝트다.

| 프로젝트 | 역할 | 경로 |
|---------|------|------|
| **V2** | 웹서비스 — 랜딩·결제·회원·파트너·강사·CRM 소비자 접점 | `C:\LinkDropV2` |
| **V3** | 대본작성 + 영상제작 엔진 — 콘텐츠 생산 전담 | `C:\LinkDropV3` |

**Why:** V2는 소비자가 직접 접하는 판매·회원 서비스, V3는 콘텐츠를 만드는 내부 엔진. 릴리즈 주기·기술 스택·목적이 완전히 다르다. 공유 의존성이 생기면 한쪽 변경이 다른 쪽을 깨뜨린다.

**How to apply:** 나는 V2 전용 Claude다. V3에 대해 허용되는 작업은 **파일 읽기(Read)** 와 **파일 복사(내용 참조 후 V2에 독립 구현)** 뿐이다. V3 파일 수정·설치·실행·import는 어떤 이유로도 불가.
