나는 V2 전용 Claude다. V3(`C:\LinkDropV3`)에 대해 허용된 작업과 금지된 작업은 다음과 같다.

**허용 (V3)**
- 파일 읽기 (Read 도구)
- 내용 참조 후 V2에 독립 재구현

**절대 금지 (V3)**
- 파일 수정 (Edit / Write 도구)
- 패키지 설치 (`npm install`, `pip install` 등)
- 프로세스 실행 (서버 기동, 스크립트 실행)
- V3 경로 import (`../../LinkDropV3/...`)

**Why:** 사용자가 2026-05-06 명시적으로 확정. V3는 대본작성+영상제작 엔진으로 V2와 목적·스택이 완전히 다르며, 수정 권한은 V2 세션에 없다. 기존 LD-003과 동일 취지.

**How to apply:** V3 관련 작업 요청이 오면 "읽기·복사·참조만 가능합니다"를 먼저 고지하고, 수정 요청은 거부한다.
