## 3. 컴포넌트 스펙: LdMascotFloat

### 파일 위치
```
apps/web/src/components/landing/LdMascotFloat.tsx
apps/web/public/img/mascot-astronaut.png
```

### Props

```ts
interface LdMascotFloatProps {
  side: 'left' | 'right'     // 등장 방향
  size?: 'sm' | 'md' | 'lg'  // sm=72px / md=120px / lg=160px (기본 md)
}
```

### 크기 토큰

| size | width | height |
|------|-------|--------|
| sm   | 72px  | 72px   |
| md   | 120px | 120px  |
| lg   | 160px | 160px  |

## 4. 레이아웃 구조

```
<div class="ld-mascot-divider">          ← h-0, position:relative, z-index:10
  <img class="ld-mascot side-right" />   ← position:absolute, right:2rem, top:-60px
</div>
```

섹션 경계 사이에 삽입하는 **높이 0짜리 divider** 방식.  
섹션 간 여백을 잡아먹지 않으면서 마스코트가 경계에 걸쳐 보임.

---

## 5. 애니메이션 스펙

### float 애니메이션 (CSS keyframes)
```css
@keyframes ld-mascot-float {
  0%, 100% { transform: translateY(0px); }
  50%       { transform: translateY(-10px); }
}
.ld-mascot { animation: ld-mascot-float 3s ease-in-out infinite; }
```

### 등장 애니메이션
- side=right: `translateX(60px) → translateX(0)`, duration 600ms, ease-out
- side=left:  `translateX(-60px) → translateX(0)`, duration 600ms, ease-out
- IntersectionObserver로 뷰포트 진입 시 트리거

### prefers-reduced-motion 대응 (필수 — LD 원칙)
```css
@media (prefers-reduced-motion: reduce) {
  .ld-mascot { animation: none; transform: none; }
}
```

| mobile (<768px) | right: 0.5rem, top: -36px, size: sm | left: 0.5rem, top: -36px, size: sm |
