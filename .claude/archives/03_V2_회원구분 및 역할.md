# 03 · V2 회원 구분 및 역할 (Role System)

> 최종 업데이트: 2026-04-25
> 관련 잠금 결정: LD-011 (`.claude/rules/LOCKED_DECISIONS.md`)

---

## 역할 체계 개요

| role | 표시 명칭 | 진입 방식 | 접근 가능 영역 |
|------|----------|----------|--------------|
| `guest` | 일반회원 | 소셜 로그인 직후 자동 부여 | 로그인 상태 유지만 (콘텐츠 없음) |
| `partner` | 파트너회원 | 1회성 결제(₩59,000) 완료 시 자동 승격 | `/member/*` + `/partner/*` |
| `gold_partner` | 골드파트너회원 | 월결제(₩29,000/월) 완료 시 자동 승격 | `/member/*` + `/partner/*` |
| `instructor` | 강사회원 | 관리자 수동 부여 | `/member/*` + `/partner/*` + `/instructor/*` |
| `admin` | 관리자 | 관리자 수동 부여 | 전체 (`/admin/*` 포함) |

> `buyer` 역할은 DB·코드 모두 삭제됨 (2026-04-25). DB CHECK 제약으로 재진입 불가.

---

## 진입 흐름

```
소셜 로그인 (Google / Kakao)
    │
    ▼
일반회원 (guest)            ← 자동 (콘텐츠 접근 불가)
    │
    ├── 1회성 결제 (₩59,000)
    │       ▼
    │   파트너회원 (partner)  ← 자동 승격
    │
    ├── 월결제 (₩29,000/월)
    │       ▼
    │   골드파트너회원 (gold_partner) ← 자동 승격
    │
    └── 관리자 수동 부여
            ▼
        강사회원 (instructor) → 관리자 (admin)
```

## 역할별 상세 권한

### 일반회원 (guest)
- 소셜 로그인 직후 자동 부여
- 콘텐츠 열람 불가
- 파트너 초대 추적 가능 (장바구니·CS 대응용)
- `/member/*` `/partner/*` 접근 시 `/?auth=1` 리다이렉트

### 파트너회원 (partner)
- ₩59,000 1회 결제 완료 시 자동 승격 (LD-011)
- **콘텐츠 열람**: V3가 생산한 강의·자료 전체
- **파트너 대시보드**: 수당 현황, 파트너 네트워크
- **추천 링크**: 초대 링크 생성·공유
- **수당 구조** (방문판매법 §22조 준수, 35% 이내):
  - 직접 판매: **₩18,700/건 (34.9%)**
  - 후원 수당: **₩9,200/월**

### 골드파트너회원 (gold_partner)
- ₩29,000/월 구독 결제 완료 시 자동 승격
- 파트너회원 권한 전체 포함
- 월결제 유지 중 골드 등급 유지, 결제 중단 시 파트너회원으로 다운

### 강사회원 (instructor)
- 파트너회원 + 골드파트너회원 권한 포함
- **강의 관리**: 강의 목록·수정·통계
- **수강생 관리**: 수강생 현황·진도 조회
- 관리자가 수동으로 부여

### 관리자 (admin)
- 전체 권한
- **회원 관리**: 역할 변경, 계정 활성/비활성
- **주문 관리**: 결제 내역, 상태 변경, 환불 처리
- **파트너 관리**: 수당 정산, 정산 완료 처리
- 관리자가 수동으로 부여

```sql
provider      text  -- 'google' | 'kakao'
first_purchase_at  timestamptz
created_at    timestamptz
```

### 역할 변경 SQL (관리자 직접 실행)

```sql
-- 파트너회원 승격 (1회 결제)
UPDATE profiles SET role = 'partner'      WHERE email = '대상이메일';

-- 골드파트너회원 승격 (월결제)
UPDATE profiles SET role = 'gold_partner' WHERE email = '대상이메일';

-- 강사회원 승격
UPDATE profiles SET role = 'instructor'   WHERE email = '대상이메일';

-- 일반회원으로 초기화
UPDATE profiles SET role = 'guest'        WHERE email = '대상이메일';
```

## 라우트 가드 (구현 예정 — AUTH-GUARD)

```
/member/*     → partner·gold_partner·instructor·admin (guest → /?auth=1)
/partner/*    → partner·gold_partner·instructor·admin (guest → /?auth=1)
/instructor/* → instructor·admin
/admin/*      → admin 전용
```
