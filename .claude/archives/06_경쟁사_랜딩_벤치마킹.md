## 1. 즉각적 ROI 시각화 — 시간 비교표

### 원본 구조
| 작업 | Before | After |
|------|--------|-------|
| 월간 콘텐츠 기획 | 4~5시간 | 5분 |
| 바이럴 영상 대본 | 2시간 | 10분 |
| 광고 카피 작성 | 1.5시간 | 3분 |
| 카드뉴스 제작 | 1시간 | 5분 |

### 왜 작동하는가
추상적 편의("쉽고 빠르다")를 **구체적 숫자**로 변환. 사용자가 자신의 상황에 즉시 대입 가능.
구체적일수록 신뢰도가 높아지고 이탈이 줄어든다.

### V2 적용 방향
- 서비스별 Before/After 비교표 섹션 추가
- 웹소설: "대본 구성 3일 → 30분"
- 영상 자동화: "편집 4시간 → 10분"
- 프롬프트 라이브러리: "카피 작성 2시간 → 3분"
- **배치 위치**: 히어로 섹션 직후, 관심을 잡자마자 바로 가치 증명

## 2. 앵커 가격 + 보너스 스태킹

### 원본 구조
```
보너스 합산 가치:  429,300원
실제 가격:          19,900원
절약:              409,400원 (95% 할인)
```

### 왜 작동하는가
사람은 **절대 가격**이 아닌 **상대 가격**으로 판단한다.
"얼마인가"가 아니라 "얼마나 아끼는가"로 결정함.
보너스가 많을수록 메인 제품의 체감 가치가 올라간다.

### V2 적용 방향
- 파트너 회원권 혜택 목록을 합산 가치로 환산해서 표기
- 예: "단독 콘텐츠 라이브러리 ₩OO만원 상당 + 파트너 수당 자격 + 추천 링크 도구 = 총 ₩OOO원 상당"
- 주의: **과장 금지** — 실제 측정 가능한 가치만 기재

## 3. 타겟 오디언스 명시 섹션

### 원본 구조
"이런 분들을 위한 제품입니다" — 4개 그룹으로 분류:
- 창업가 및 비즈니스 대표
- 마케팅 전문가
- 콘텐츠 크리에이터
- 학생 및 AI 관심자

각 그룹에게 다르게 말을 건다.

### 왜 작동하는가
"나를 위한 제품"이라는 인식이 생기면 이탈률이 급격히 낮아진다.
광범위한 대상이어도 분류해서 보여주면 각각이 "나한테 맞는 것"으로 느낌.

### V2 적용 방향
- 랜딩 히어로 이후 "이런 분께 딱 맞습니다" 섹션
- V2 타겟: 직장인 부업 희망자 / 콘텐츠 크리에이터 / 영상 제작자 / 소상공인
- 각 그룹마다 **구체적인 상황 묘사** — "퇴근 후 2시간으로 부업 시작하고 싶은 40대"

## 4. FAQ — 반론 미리 차단

### 원본 구조
결제 섹션 직전에 9개 FAQ 배치.
고객 지원 목적이 아닌 **머릿속 구매 저항 제거** 목적.

### 왜 작동하는가
사람은 결제 직전에 가장 많이 이탈한다.
그 순간 떠오르는 질문("정말 쓸 수 있을까?", "환불은 되나?")에 미리 답하면 이탈이 줄어든다.

### V2 적용 방향
- CTA 버튼 바로 위에 FAQ 섹션 배치
- 예상 반론 목록:
  - "나는 AI를 잘 모르는데 할 수 있나요?" → 초보 친화적 설명
  - "수당은 언제부터 받나요?" → 즉시 시작 강조
  - "한 달에 얼마나 벌 수 있나요?" → 현실적 사례 제시
  - "지금 당장 시간이 없는데요?" → 1일 1시간으로 시작하는 방법

## 5. ❌ 환불 보장 배지 — 참조 불가 (사기 패턴)

### 실제 확인된 사실
직접 구매 후 환불 요청 → 실질적으로 환불 불가 시스템.
"7일 환불 보장"은 구매 저항을 낮추는 미끼 문구로, 실제 이행 의사 없음.

### 왜 위험한가
- 실제로 환불이 되는 시스템이라면 정당한 신뢰 요소
- 이행 불가 약속을 내거는 것은 소비자 기만 — 법적으로도 문제
- V2에 동일 방식으로 적용하면 브랜드 신뢰를 스스로 파괴하는 행위

### V2 방향 (대안)
환불 보장 문구 자체가 아니라 **처음부터 리스크가 없는 구조**를 만든다:
- 유료 전환 전 무료 체험 가능한 콘텐츠 제공
- 과장 없는 실제 내용으로 기대치 관리
- 환불 약속 대신 **구매 전 충분한 정보 제공**으로 분쟁 원천 차단

## 6. 창업자 퍼소나 — 인간적 신뢰 구축

### 원본 구조
- 이름·사진·약력 공개
- "글로벌·아시아 수천 명을 도왔다" 스토리
- 결제 페이지가 아닌 제품 설명 중간에 등장

### 왜 작동하는가
사람에서 사람으로 사는 느낌. 회사 로고보다 얼굴이 신뢰를 만든다.
특히 디지털 제품에서 "누가 만들었나"는 구매 결정의 주요 변수.

### V2 적용 방향
- 파트너 모집 페이지에 팀/창업자 스토리 섹션 추가
- 과장 없이 실제 경험 기반으로 작성
- 사진·이름·구체적 숫자(몇 명 도움, 몇 년 경험)

## 7. 긴급성 언어 (Urgency Copy)

### 원본 구조
- "게릴라 특가" — 한정 느낌
- "지금 바로" — 즉시 행동 유도
- 현재 가격이 곧 오를 것 같은 뉘앙스

### 왜 작동하는가
인간은 손실 회피 본능이 있다. "지금 안 사면 손해"라는 프레임이 전환을 올린다.
단, **거짓 긴급성**은 신뢰를 파괴한다.

### V2 적용 방향
- 실제 조건 기반 긴급성만 사용
- "이번 달 파트너 모집 O명 한정" (실제라면)
- "얼리버드 가격 — 베타 기간에만 적용"
- 거짓 카운트다운 타이머 절대 금지
