### 9. WordPress 자동 배포 (동의 기반 플랫폼)
- **현황**: 미구현. 설계 논의 완료 (2026-04-15)
- **전략**: V3 사용자가 동의 시 생성 콘텐츠를 LinkDrop WordPress 플랫폼에 자동 발행
- **필요 작업**:
  - `wordpress_service.py` 신규 작성 (WordPress REST API 연동)
  - `v3_series.settings`에 `wordpress_publish: bool` 동의 플래그 추가
  - 파이프라인 마지막 스텝에 wordpress 발행 게이트 추가
  - 시리즈 생성 UI에 동의 opt-in 체크박스 추가
  - 작가 크레딧(사용자명 + 본인 채널 링크) 자동 삽입
- **설계 원칙**:
  - opt-in(명시적 동의)만 허용 — opt-out 방식 금지
  - 동의 철회 기능 필수
  - 3-Judge 점수 기준선 미달 시 발행 보류 (품질 필터)
- **비고**: 사용자 콘텐츠 집약 → SEO 자산 플랫폼화 → CAC 0원 신규 유입 구조
