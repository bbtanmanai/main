# 105 — 이미지 저장 정책 & 금지 사항

## 이미지 저장 정책 — 단계별 기준

### 개발 단계 (현재) — 로컬 우선
- `output/` 또는 `tmp/` 폴더에 로컬 저장 후 경로로 참조해도 무방
- R2 업로드는 선택 사항 (검증 완료된 결과물만 올림)
- **git에 이미지 커밋 금지** — `.png/.jpg/.mp3/.mp4/.ttf` 등 바이너리는 커밋하지 않음

### 프로덕션 전환 시 — R2 필수
- 로컬 경로 참조 제거 → `R2_PUBLIC_URL/{경로}` 형태로 전환
- R2 버킷: `flux-bg-library` (`R2_BUCKET`) / `linkdrop-assets` (`R2_ASSETS_BUCKET`)

---

## 금지 사항

- API 키·토큰·시크릿을 코드나 설정 파일에 직접 작성 금지
- `git push --force` 금지 / `--no-verify` 플래그 사용 금지
- UI에서 "세션" 표현 금지 → "AI 연결", "AI 사용 가능 시간" 등 일반 언어 사용
- 임시 스크립트를 루트에 생성 금지 → `tmp/` 폴더에만 생성, 사용 후 삭제
