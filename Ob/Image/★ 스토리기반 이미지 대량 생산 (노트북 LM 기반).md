
유료사용자 : Gemini 3 Pro 기반 모델
무료사용자 : **Gemini 3 Flash**  or  Gemini 3 모델

1. 3000자 이내로 스토리 작성 (성우 기준 7분 정도 - )
2. Create Audio and Video Overviews from 이라는 팝업창이 열립니다.
3. 이 팝업창의 우측 하단에 'Copy Text' 버튼을 클릭합니다. 
4. 텍스트 스토리를 붙혀넣기
5. 우측 하단에 'Insert' 버튼을 누릅니다.
6. 아래쪽에 있는 슬라이드 덱에 있는 연필 모양의 아이콘을 클릭합니다.
7. 'Slide deck you want to create' 창에 다음과 같이 입력합니다. 

```
각각의 슬라이드가 키프레임이 되도록 해주고 슬라이드에는 텍스트를 넣지 말아줘. 
슬라이드 스타일은  할리우드의 블록버스터 SF 영화 스타일로 해줘.
```

```
각각의 슬라이드가 키프레임이 되도록 해주고 슬라이드에는 텍스트를 넣지 말아줘. 
슬라이드 스타일은 다크 시네마틱 실사풍 스타일로 해줘.
```


```
각각의 슬라이드가 키프레임이 되도록 해주고 슬라이드에는 텍스트를 넣지 말아줘.
슬라이드 그림 스타일은 일본 SF 애니메이션으로 해줘.
```

```
각각의 슬라이드가 키프레임이 되도록 해주고 슬라이드에는 텍스트를 넣지 말아줘. 
슬라이드 그림 스타일은 타무라 시게루의 일러스트레이션으로 해주고, 
컬러를 몇 가지로만 제한해서 타무라 시게루 특유의 인상적인 분위기를 연출해줘.
```

```
 각각의 슬라이드가 키프레임이 되도록 해주고 슬라이드에는 텍스트를 넣지 말아줘.
 슬라이드 그림 스타일은 수묵화로 해주고 붓의 힘과 속도감이 느껴지는 역동적인 필치와 농담의 변화를 풍부하게 표현해줘.
```


 9. 제미나이 프로 사용자 기준 약 15분, 무료 사용자는 약 30분 기다리면 영화의 한 장면 같은 이미지들로 15장의 슬라이드가 완성됩니다.
10. 제미나이 프로 사용자는 일 10편 ~ 20편의 슬라이드,  무료 사용자는 하루 3편
 11. 생성된 슬라이드 우측에 점 세 개를 클릭하면 펼쳐지는 메뉴에서 'Start Slide Show'를 선택
 12. 전체 화면에 슬라이드가 꽉 차게 보여줍니다. 
 13. 화면을 클릭하면 다음 페이지로 넘어갑니다. 
 14. 키보드에 프린트 스크린 키를 눌러 한 장씩 슬라이드를 캡처해서 15장을 저장합니다. 
 15. 이 이미지들은 제작할 단편 영화의 키프레임이 됩니다.
