# 🛠️ 지식 정제 에이전트 (Knowledge Refiner)

## 개요
수집된 원재료(RAW) 데이터 중 영상 콘텐츠로서의 가치가 높은 소재를 선별하는 독립형 정제 스킬입니다. AI 스코어링을 통해 7.0점 이상의 고품질 지식(GOLD)만 골라내어 '지식 서랍장'에 저장합니다.

## 핵심 기능
- **AI 스코어링**: LLM을 활용하여 원문의 정보 밀도, 영상화 적합성, 타겟팅 가치를 분석.
- **고품질 선별**: 기준 점수(7.0점) 이상만 골드로 승격하여 파이프라인의 품질 유지.
- **데이터 저장**: 정제된 데이터를 `packages/data/02_지식_서랍장/{Theme}` 폴더에 독립 저장.

## 사용법 (AI 지침)
- 수집 단계(No.01)가 완료된 후, 본 스킬을 호출하여 원재료를 정제하세요.
- 입고된 `GOLD_*.json` 파일들은 이후 '영상 설계(No.03)' 단계의 입력값으로 사용됩니다.

## 제약 사항
- OpenAI API 키가 환경변수(`OPENAI_API_KEY`)에 설정되어 있어야 합니다.
- 스코어링 기준은 `logic.py` 내의 `threshold` 파라미터로 조절 가능합니다.
