# 🛠️ 출고 동기화 에이전트 (Drive Syncer)

## 개요
최종 생산된 지식 콘텐츠(GOLD) 및 영상 설계도를 구글 드라이브(G-Drive)의 지정된 테마 폴더로 업로드하여, 외부 서비스 및 노트북LM과 실시간 동기화하는 독립형 배송 스킬입니다.

## 핵심 기능
- **테마별 지능형 업로드**: 건강, 수익, 기술 등 5대 테마에 맞춰 드라이브 폴더를 자동으로 매핑하여 배송.
- **배송 상태 관리**: 업로드 완료된 파일에 `_SYNCED` 라벨을 부여하여 중복 전송을 방지하고 배송 이력을 관리.
- **멀티 에셋 동기화**: JSON 데이터뿐만 아니라 이미지, 영상 설계도 등 다양한 형태의 결과물을 구글 클라우드로 배송 가능.

## 사용법 (AI 지침)
- 모든 공정(No.01~No.05)이 완료되고 최종 결과물을 외부로 출고해야 할 때 본 스킬의 `sync_to_drive()`를 호출하세요.
- 구글 드라이브에 업로드된 자료는 노트북LM이 즉시 학습하여 '삼위일체 지식 베이스'를 강화하는 데 사용됩니다.

## 제약 사항
- 구글 API 인증 파일(`token.json`)이 유효해야 합니다.
- 드라이브 폴더 ID는 환경변수(`DRIVE_FOLDER_*`)를 통해 사용자 정의가 가능합니다.
