# 🛠️ 지식 조립 에이전트 (Knowledge Assembler)

## 개요
정제된 골드(GOLD) 지식 콘텐츠를 실제 영상 제작을 위한 `Rule 104` 표준 기반 대본으로 변환하는 독립형 스킬입니다. 시니어의 가독성을 최우선으로 고려하여 장면을 설계합니다.

## 핵심 기능
- **Rule 104 대본 설계**: 시니어 가독성(45자 이내) 및 효과음(SE) 타이밍을 반영한 정교한 장면(Scene) 구성.
- **시각적 가이드 생성**: 각 장면과 어울리는 고품질 이미지 생성을 위한 영어 프롬프트 자동 작성.
- **오디오 연출 설계**: BGM 대신 효과음(Ding, Whoosh, Pop)을 활용한 오디오 집중형 연출 설계.

## 사용법 (AI 지침)
- 지식 분류(No.03)가 완료된 골드 지식을 실제 제작 패키지로 만들 때 본 스킬의 `generate_script()`를 호출하세요.
- 생성된 `SCRIPT_*.json` 파일은 이후 '영상 설계(No.05)' 단계에서 오팔 또는 리모션 엔진의 입력값으로 사용됩니다.

## 제약 사항
- OpenAI API 연동이 필요하며, `gpt-4o-mini` 이상의 모델 사용을 권장합니다.
- 복잡한 지식은 AI가 여러 장면으로 분할 설계하며, 총 길이는 1분 내외로 최적화됩니다.
