# 🛠️ 네이버 브랜드커넥트 크롤링 (Naver BrandConnect Crawling)

## 개요
네이버 브랜드 커넥트 및 스마트스토어 상세 페이지에서 고화질 상품 이미지와 텍스트 데이터(가격, 리뷰 등)를 추출하는 독립형 크롤링 도구입니다.

## 핵심 기능
- **고화질 이미지 수집**: `m1000_pd` 규격의 최대 해상도 이미지 자동 확보.
- **데이터 정제**: AI 대본 생성을 위한 가격, 할인율, 실사용 리뷰 추출.
- **독립 저장**: `packages/data/01_재료_하차장/NAVER/{키워드}` 폴더에 모든 에셋을 패키징하여 저장.

## 사용법 (AI 지침)
- 사용자가 네이버 쇼핑 링크를 주면 본 스킬을 호출하여 브랜드 커넥트 데이터를 크롤링하세요.
- 추출된 `metadata.json`의 내용을 바탕으로 영상 대본(Rule 104)을 작성하세요.

## 제약 사항
- 크롤링 방지 로직을 피하기 위해 Headless Chrome을 사용합니다.
- 과도한 요청은 차단될 수 있으므로 배칭 처리를 권장합니다.
