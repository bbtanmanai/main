# 🛠️ 콘텐츠 소재탐사 에이전트 (Vault Source Fetcher)

## 개요
네이버 뉴스, 구글 뉴스 RSS 및 웹 본문 추출을 통해 고순도 소재를 자동으로 수집하는 독립형 스킬입니다. 링크드랍의 5대 핵심 Vault(건강, 수익, 기술, 자기계발, 마케팅) 테마에 맞춰 정보를 분류하여 입고합니다.

## 핵심 기능
- **테마별 지능형 수집**: 5대 핵심 키워드에 기반한 최신 트렌드 뉴스 수집.
- **고정밀 본문 추출**: `trafilatura`를 활용한 광고 없는 순수 텍스트 추출.
- **데이터 구조화**: 수집된 본문을 JSON 형태로 `packages/data/01_재료_하차장/{Theme}` 폴더에 독립 저장.

## 사용법 (AI 지침)
- 특정 주제의 최신 트렌드가 필요할 때 본 스킬의 `fetch_vault_updates()`를 호출하세요.
- 입고된 `RAW_*.json` 파일들은 이후 '지식 심층분석' 단계의 입력값으로 사용됩니다.

## 제약 사항
- 네트워크 상태에 따라 본문 추출이 실패할 수 있습니다.
- 대량 수집 시 타겟 사이트의 차단을 방지하기 위해 딜레이를 권장합니다.
