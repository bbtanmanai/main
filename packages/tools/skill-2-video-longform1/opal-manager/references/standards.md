# Opal Node Standards

Reference for ensuring all rendering nodes comply with the LinkDropV2 senior-friendly production standards.

## 1. TextNode
- **Font**: `Gmarket Sans`
- **Minimum Size**: `72pt`
- **Safe Area**: Bottom 15%
- **Background Bar**: Black, Opacity 0.4

## 2. AudioNode
- **BGM Volume**: `0.0`
- **Voice Tone**: `ko-KR-Wavenet-D`
- **SE Themes**:
    - `scene_open_whoosh`
    - `question_ding`
    - `highlight_pop`

## 3. MotionNode
- **Primary Effect**: `Ken-Burns-Zoom` (Slow)
- **Shake intensity**: `0.0` (Absolute)

## 4. GradeNode
- **LUT Policy**: Based on Scene Mood (standardized in `opal_service.py`)
