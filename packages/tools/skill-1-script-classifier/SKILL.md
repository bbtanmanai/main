# 🛠️ 지식 테마분류 에이전트 (Script Classifier)

## 개요
정제된 지식 콘텐츠를 분석하여 서비스 운영에 최적화된 5대 핵심 카테고리(건강, 재테크, 기술, 지혜, 라이프스타일)로 자동 분류하는 독립형 스킬입니다.

## 핵심 기능
- **문맥 기반 분류**: LLM을 활용하여 단순 키워드 매칭이 아닌 본문 전체의 맥락을 파악하여 가장 적합한 테마를 결정.
- **표준 카테고리 매핑**: 링크드랍의 5대 표준 카테고리 규격 준수.
- **자동 라벨링**: 콘텐츠 메타데이터에 테마 정보를 추가하여 후속 공정(디자인, 제작)이 테마별로 다르게 가동될 수 있는 지침을 제공.

## 사용법 (AI 지침)
- 지식 정제(No.02)가 완료된 후, 콘텐츠의 성격에 맞는 템플릿을 고르거나 테마별 서랍장에 저장해야 할 때 본 스킬을 호출하세요.
- 분류된 결과는 `packages/data/02_지식_서랍장/{Theme}` 아래에 체계적으로 저장됩니다.

## 제약 사항
- OpenAI API 키가 환경변수(`OPENAI_API_KEY`)에 설정되어 있어야 합니다.
- 텍스트 길이가 너무 짧으면 분류 정확도가 떨어질 수 있습니다.
