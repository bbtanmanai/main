
# 시나리오 기반 프레임별 이미지 프롬프트 앱 설계 (Google AI Studio / Gemini)

## 1. 목표

- 시나리오 텍스트와 2명의 배우 캐릭터 시트 이미지를 입력하면  
  **각 프레임(컷)마다 이미지 생성용 상세 프롬프트**를 자동으로 만들어 주는 앱.
- 결과물은 **텍스트 프롬프트만** 출력하며, 이미지/비디오를 직접 생성하지 않는다.
- 사람(사용자)이 중간에 프롬프트를 검토·수정할 수 있는 워크플로우를 포함한다.

---

## 2. 전체 워크플로우 개요

1. 입력 단계
   - 시나리오 텍스트 (씬/샷 단위)
   - 배우 2인의 캐릭터 시트 이미지 (얼굴, 전신 등 여러 장)[cite:6][cite:8]
2. 분석 단계 (Gemini API)
   - 시나리오를 **씬 → 프레임(이미지 단위)**로 분해
   - 각 프레임마다 등장 인물, 상황, 구도, 감정, 배경, 스타일을 추출[web:18]
3. 초안 생성 단계
   - Gemini가 프레임별 **이미지 프롬프트 초안(draft)**을 JSON 형식으로 생성[web:18][web:20]
4. 중간 점검 단계 (사용자 UI)
   - 엑셀/테이블 형태 화면에서 프레임별 프롬프트를 검토·수정·확정[file:29]
5. 출력 단계
   - 확정된 프롬프트를 JSON/CSV/엑셀로 내보내어  
     외부 이미지 생성 모델(Imagen, 기타 이미지 AI)에 사용[web:16][web:25]

---

## 3. 모델/플랫폼 선택

- 플랫폼: **Google AI Studio → Gemini API**.[web:18][web:37]
- 모델:
  - 멀티모달 이해용: `gemini-2.0-flash` 또는 `gemini-2.0-pro` (텍스트+이미지 입력).[web:20]
- 사용 방식:
  - Gemini는 **“프롬프트 텍스트 초안 생성기”**로만 사용한다.
  - 최종 이미지는 Imagen 등 별도 이미지 생성 모델에서 생성.[web:16][web:25]

---

## 4. 시스템 인스트럭션(역할 정의)

Gemini 호출 시 고정으로 넣는 시스템 프롬프트 개념.

### 4.1 역할

- 너는 “시나리오 기반 이미지 프롬프트 엔진”이다.
- 입력:
  - 시나리오 텍스트
  - 2명의 캐릭터 시트 이미지 (배우 외형 레퍼런스)
- 출력:
  - 시나리오를 프레임 단위로 나누고,
  - 각 프레임마다 이미지 생성용 프롬프트를 **JSON 형식**으로만 출력한다.
- 이미지나 비디오를 생성하거나, 마크다운/자연문을 따로 쓰지 않는다.  
  오직 JSON만 출력한다.[web:18]

### 4.2 캐릭터/스타일 규칙

사용자가 이미 정의한 3원칙을 반영.[cite:6][cite:8][cite:10]

1. 이미 등장한 캐릭터의 얼굴, 체형, 헤어, 기본 의상 콘셉트를 바꾸지 않는다.
2. 전체 화풍과 스타일(조명, 색감, 분위기)을 유지한다.
3. 물리적 시간·공간·인체에 대해 과장되거나 불가능한 묘사를 하지 않는다.

추가 규칙:

- 각 프롬프트에서 캐릭터는 **캐릭터 시트 이미지와 일관되게** 묘사한다.
- 새 문신·흉터·장신구를 멋대로 만들지 않는다 (시나리오에 명시된 경우만 예외).[cite:8][cite:9]
- 이미지 한 장에만 해당하는 내용으로 묘사하고, 카메라 움직임이나 시간 경과 표현은 피한다.[web:16][web:25]

## 5. 데이터 구조 (JSON 스키마)

### 5.1 전체 구조

```json
{
  "project_id": "example_project",
  "characters": [
    {
      "id": "character_GuanYu",
      "name": "관우",
      "reference_files": ["file_id_guan_face", "file_id_guan_body"],
      "description": "..."
    },
    {
      "id": "character_HuaTuo",
      "name": "화타",
      "reference_files": ["file_id_hua_face", "file_id_hua_body"],
      "description": "..."
    }
  ],
  "scenes": [
    {
      "scene_id": "S03",
      "title": "관우의 막사",
      "frames": [
        {
          "frame_id": "s03c07b01",
          "timecode": "00:10-00:17",
          "summary_ko": "관우 등이 드러난 채 화타가 상처를 살피는 첫 컷",
          "prompt_ko_draft": "...",
          "prompt_en_draft": "...",
          "prompt_ko_final": null,
          "prompt_en_final": null,
          "characters": [
            {
              "character_id": "character_GuanYu",
              "role_in_frame": "main",
              "pose_description": "...",
              "expression_description": "..."
            },
            {
              "character_id": "character_HuaTuo",
              "role_in_frame": "supporting"
            }
          ],
          "camera": {
            "shot_size": "medium close-up",
            "angle": "slightly low angle"
          },
          "style": {

          }
        }
      ]
    }
  ]
}

```

- `prompt_*_draft` : Gemini가 만든 초안.
    
- `prompt_*_final` : 사용자가 검수·수정한 확정본.
    
- `timecode`는 이미지라 해도 “씬 내 위치”를 기억하기 위한 메타정보로 유지.
