> [!info]
> #### 1그룹: 핵심 제작 로직 (Core Production Logic)
> 
> Primary Key Visual (PKV, 주요 키 비주얼): 특정 시퀀스의 시각적 시발점이 되는 마스터 샷입니다. (구성: 장면 + 캐릭터 행동 + 선택된 복장)
> Visual Reference Anchor (VRA, 시각적 참조 앵커): PKV를 구성하기 위해 호출되는 개별 소스 에셋입니다. 
> Master Code (마스터 코드): SxxCxxBxx 형태의 고유 주소입니다. (예: S01C01B03 - 시나리오 01, 컷 01, 블록 03)
> Character Consistency (캐릭터 일관성): 인물이 영상 내내 동일한 외형(이목구비, 골격)을 유지하는 상태입니다.
> Environment Persistence (환경 지속성): 배경의 광원, 색감, 물리적 구조가 컷이 바뀌어도 변함없이 유지되는 성질입니다.
> 
> #### 2그룹: 캐릭터 및 자산 관리 (Character & Asset)
> 
> Exclusive Clothing (전용 복장): 특정 캐릭터만 입는 고유 의상입니다. (예: 유비 전용 평상복)
> Signature Items (시그니처 아이템): 인물을 상징하는 핵심 소품입니다. (예: 청룡언월도, 장비의 뱀창)
> Asset_Reference_Injection (자산 참조 주입): DB에 저장된 실제 이미지 경로를 AI에게 직접 전달하여 그대로 그리게 하는 로직입니다.

#### 이미지 제작 플로우

```
① 나노에서 Seed 고정 → 외형 기초 다지기 (다량 제작으로 원하는 컨셉 찾기)
② 미드저니에서 Seed 고정 → 스타일·변형·고퀄 제작
③ 둘은 "같은 Seed 숫자"지만 역할은 각자 독립적으로 수행
   방식이 캐릭터 안정성 / 비용 절감 / 대량 생성 / 스타일 유지 모두에서 최적입니다.
```

```
studio lighting: 선명하고 깔끔한 느낌
```
